# Task-2
Perform DML in SQL like insert , update and delete data from a table.


DML : Data Manipulation Language used to change the structure of data in database.
1.Insert 
-used to inserting a data into table 
syntax:
insert into table_name (colum_names) values (records);
example:
insert into customers (name,email) values ("Sanika","jadhavsanika874@gmail.com");


2.update
-used to updating a specific records in table
syntax:
update table_name set coulmn_name=value where condition;
example:
update table customers set email='jadhavsanika874@gmail.com' where id=1;


3.delete
-used to deleting a reo in table
syntax:
delete from table_name where condition;
example:
delete from customers where id=4;
